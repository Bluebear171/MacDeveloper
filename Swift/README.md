# Swift 学习笔记

Swift是一种支持多编程范式和编译式的编程语言，是用来撰写 macOS/OS X、iOS、watch OS 和 tvOS 的语言之一， 2014年，在苹果开发者年会（WWDC）发布。

这一份资料(笔记)基于 Swift 3.0.2 版本，不同版本可能会出现部分语法不兼容问题。
